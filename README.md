## Build and run:

```
yarn
yarn build
yarn dev
```

Pages:
 - http://localhost:3000/ - Login form (admin@admin.com, admin)
 - http://localhost:3000/counter
 - http://localhost:3000/rickandmorty - Rick and morty statechart

Ensure you enable popups in your browser!
